# SPAPP — Fitness Booking App
## Project Overview

**Data creazione:** 2026-02-20
**Stato:** Planning Phase

---

## Descrizione del Progetto

SPAPP è un'applicazione mobile per la gestione e prenotazione di sessioni di allenamento fitness. L'app permette agli utenti di:

- Prenotare sessioni specifiche di allenamento
- Visualizzare il proprio profilo privato e storico
- Accedere alla programmazione di allenamento personalizzata
- Visualizzare gli esercizi in modo interattivo (animazioni che mimano i movimenti)
